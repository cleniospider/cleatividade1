Resumo das Principais Funcionalidades: Flexbox e Grid

Página criada para apresentar as principais propriedades de cada um.

FLEXBOX

O que é: Sistema de layout em uma direção (linha ou coluna).

Principais Propriedades:

  • flex-direction – define direção dos itens.
  • justify-content – alinha no eixo principal.
  • align-items – alinha no eixo cruzado.
  • flex-wrap – permite quebra de linha.
  • flex-grow / flex-shrink – controla expansão e redução.
  • order – muda a ordem dos itens.

CSS GRID

O que é: Sistema de layout em duas dimensões (linhas e colunas).

Principais Propriedades:

  • grid-template-columns / rows – define estrutura.
  • grid-template-areas – cria áreas nomeadas.
  • gap – espaçamento entre células.
  • grid-column / grid-row – posiciona itens.
  • repeat / minmax / auto-fit – ajuda no responsivo.
